Homework

  • Create 2 files - an HTML and a CSS file
  • Your task is to demonstrate your knowledge by importing a functional CSS stylesheet into your HTML document.
  1. Create an HTML document with at least 10 unique tags within the body
    1. At least 8 elements must have unique classes and at least 3 must have unique IDs
  2. In your CSS stylesheet, create rulesets for each of your 5 classes and 2 IDs. In addition, use the wildcard operator and at least 2 tag selectors (h1, p, etc…)
    1. Every ruleset must have at least 3 unique CSS rules (with the exception of the wildcard and tag selectors which must have at least one
    2. In total, at least 15 unique CSS rules must be used
    3. At least 2 rulesets must have pseudo-classes, with at least 1 noticeable change
  • Aim to use as many different CSS properties as possible, spanning as many types or “buckets” as you can. Feel free to explore and use any interesting CSS properties!
  • Bonus Challenge: try to find useful CSS properties the teachers have never seen before.
